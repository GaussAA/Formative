## 🎯 使用示例

### 基础操作

```bash
# 启动
pnpm ops:start

# 停止
pnpm ops:stop

# 健康检查
pnpm ops:health

# 交互菜单
pnpm ops
```

### 高级操作

```bash
# 完整清理
pnpm ops:clean --all

# 重装依赖
pnpm ops:reinstall --verify

# 完整维护
pnpm ops:stop && pnpm ops:clean --all && pnpm ops:reinstall && pnpm ops:start
```
